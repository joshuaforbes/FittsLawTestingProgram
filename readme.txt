Compilation Instructions:

	The program is built using Java FX thus Java 1.8 is required to compile the program and run the program.
	
	To compile simply use the makefile provided and type in the following command:
		
		# make compile
		
	Then to run simply use;
	
		# make run
		
	To clean up afterwards use:
	
		# make clean
		
Do not clean and then run, the program will fail to run.
